compilefile.py is used to compile java source and junit tests to generate java class
controller.py is the beginning of the whole project
listdir.py is used to generate a dictionary tree of the project
runbackup.py is a backup file. in this version, the project can run the users' own test on their own java source
runtest.py is a new version of running tests. It can allow different users to run their tests on different java files
